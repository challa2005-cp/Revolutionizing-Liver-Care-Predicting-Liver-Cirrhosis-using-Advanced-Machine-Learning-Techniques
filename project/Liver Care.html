Revolutionizing Liver Care

Predicting Liver Cirrhosis using Advanced Machine Learning Techniques

Introduction

Liver cirrhosis is a progressive disease that can be life-threatening if not detected early. Leveraging machine learning, we aim to predict cirrhosis risk using patient data, enabling timely intervention and personalized care.

Methodology

  • Data Collection: Clinical and biochemical data from liver patients.
  • Preprocessing: Handling missing values, normalization, and feature selection.
  • Modeling: Using algorithms like Random Forest, SVM, and Neural Networks.
  • Evaluation: Accuracy, precision, recall, and ROC-AUC metrics.

Results

Our models achieved over 90% accuracy in predicting cirrhosis, with Random Forest showing the best performance. Visualizations and detailed metrics will be added soon.
